#### 2. 【一次成功】的非流式调用代码

这是最稳定、最简单的入门代码。它会一次性返回完整的Qwen3回答。

python
from openai import OpenAI

# 步骤A：创建客户端，只修改base_url和api_key
client = OpenAI(
    base_url="https://www.qianjuai.com/v1",
    api_key="sk-你的千聚Key放在这里" # 请替换成你注册后拿到的Key
)

# 步骤B：发起请求
response = client.chat.completions.create(
    model="Qwen3-235B-A22B", # 关键步骤，这里指定Qwen3模型名称
    messages=[
        {"role": "user", "content": "请用Python写一个冒泡排序的代码"}
    ],
    temperature=0.7,
    max_tokens=2048
)

# 步骤C：提取结果
print(response.choices[0].message.content)
print("-------------------")
print(f"模型回答完毕，共消耗: {response.usage.total_tokens} tokens")


**代码解释（防坑专用）：**

-   **`base_url`**： 必须填写 `https://www.qianjuai.com/v1`。这是[千聚api聚合平台](https://www.qianjuai.com/)提供的专线。很多人的错误是填写成官网域名或者忘记加 `/v1` 后缀，导致404。
-   **`model`**： 必须准确填写 `Qwen3-235B-A22B`。如果填写 `qwen-max` 或其他过时名称，会报错或调用到不同版本的模型。请在千聚官网的模型列表里查看当前最新的Qwen3模型ID。
-   **`temperature` 和 `max_tokens`**： 这是标准参数。推荐温度设为0.7，最大Token设为2048，可以避免生成太长或太随机的回答。

#### 3. 支持流式输出的进阶代码（适合ChatBot场景）

如果你想做交互式聊天，流式输出是必不可少的。下面这段代码让你能实时看到Qwen3的思考过程。

python
from openai import OpenAI

client = OpenAI(
    base_url="https://www.qianjuai.com/v1",
    api_key="sk-你的千聚Key放在这里"
)

stream = client.chat.completions.create(
    model="Qwen3-235B-A22B",
    messages=[
        {"role": "user", "content": "请给我一个提高代码效率的建议"}
    ],
    stream=True,  # 启动流式模式
    temperature=0.7
)

# 逐块输出
for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")


**关键避坑点：** 很多人在流式模式下忘记处理 `chunk.choices[0].delta.content is not None` 这个判断。如果不加这个判断，会在输出流末端（当 `delta` 为空时）抛出 `AttributeError` 异常。请务必加上！

#### 4. 多轮对话（Memory）的正确实现

真正的应用需要上下文。这里演示如何手动维护历史消息。

python
from openai import OpenAI

client = OpenAI(
    base_url="https://www.qianjuai.com/v1",
    api_key="sk-你的千聚Key放在这里"
)

# 初始化消息列表
messages = [
    {"role": "system", "content": "你是一名资深Python工程师，请用简洁专业的语言回答问题。"},
    {"role": "user", "content": "什么是装饰器？"}
]

# 第一轮对话
response = client.chat.completions.create(model="Qwen3-235B-A22B", messages=messages, stream=True)
# (为了演示简洁，这里使用非流式)
response = client.chat.completions.create(model="Qwen3-235B-A22B", messages=messages, stream=False)
assistant_reply = response.choices[0].message.content
print("AI:", assistant_reply)

# 将上下文加入消息列表
messages.append({"role": "assistant", "content": assistant_reply})
messages.append({"role": "user", "content": "刚才你提到装饰器，能给我一个实际应用的例子吗？"})

# 第二轮对话
response = client.chat.completions.create(model="Qwen3-235B-A22B", messages=messages, stream=False)
print("AI:", response.choices[0].message.content)


**避坑提示：** [千聚api聚合平台](https://www.qianjuai.com/)完全兼容OpenAI的Message格式。注意，如果你打算长期保存对话，请控制 `messages` 列表长度。Token越长，消耗越多。一般建议保留最近10-20轮对话。
