## Qwen3 API Python 调用示例

我现在就带你写一段实际的 Python 代码，绝对简单，100% 可运行。

**第二步：写调用代码**

打开你的 Python 文件（比如 `test_qwen3.py`），复制粘贴下面这段代码。除了改你的 API key 和模型名称，其他都不用动。

python
import openai

# 设置千聚API的 base_url 为兼容地址
openai.base_url = "https://www.qianjuai.com/v1"

# 设置你的 API key
# 注意：把 "sk-xxxxx" 替换成你在千聚控制台生成的 API key
openai.api_key = "sk-xxxxx"  

# 定义具体的模型名称，这里用 Qwen3 满血版
model_name = "Qwen3-235B-A22B"  # 以千聚官方公布的模型名为准

def call_qwen3(prompt):
    """
    调用 Qwen3 模型的简单示例函数
    """
    try:
        response = openai.chat.completions.create(
            model=model_name,
            messages=[
                {"role": "system", "content": "你是一个有智慧的AI助手。"},
                {"role": "user", "content": prompt}
            ],
            max_tokens=1024,   # 控制回复长度
            temperature=0.7,   # 控制生成内容的随机性，0-2之间
            stream=False       # 设定为False表示一次性返回全部结果
        )
        # 提取并打印回复内容
        reply = response.choices[0].message.content
        print("Qwen3 回复：", reply)
        return reply
    except Exception as e:
        print("调用失败，错误信息：", str(e))

if __name__ == "__main__":
    # 测试交互
    while True:
        user_input = input("请输入你的问题（输入'q'退出）：")
        if user_input.lower() == 'q':
            break
        call_qwen3(user_input)
