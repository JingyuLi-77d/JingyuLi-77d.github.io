### 报错2：`openai.AuthenticationError: Incorrect API key provided`

**原因**：你的 API Key 写错了，或者已经失效。

**解决方案**：
1.  **重新复制**：登录千聚后台，重新复制你的 API Key，粘贴到代码里，注意不要有多余的空格。

3.  **检查代码**：确认 `api_key` 变量赋值是否正确，不要写成了 `api-key` 或其他格式。

### 报错3：`openai.BadRequestError: Error code: 400 - {'error': {'message': 'The model `o3-mini` does not exist...'}}`

**原因**：千聚上暂时没有这个模型，或者模型名称写错了。

1.  **确认模型支持**：千聚平台支持 500+ 模型，o3-mini 是包含在内的。但如果后台有临时维护，可能会显示不了。你可以登录千聚平台查看在线模型列表，确认一下状态。

**原因**：你的账户余额用完了，超出了当前的速率限制。
