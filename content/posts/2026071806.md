### 第三步：写测试代码

以下是一个完整的Java示例，用于调用GPT-4o模型完成对话：

java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.time.Duration;

public class QianJuAIDemo {
    public static void main(String[] args) throws Exception {
        // 1. 创建HttpClient实例
        HttpClient client = HttpClient.newBuilder()
                .connectTimeout(Duration.ofSeconds(10))
                .build();

        // 2. 构建请求体
        // 请确保替换 YOUR_API_KEY_HERE 为你在千聚申请的API Key
        String apiKey = "YOUR_API_KEY_HERE";

        String requestBody = """
                {
                    "model": "gpt-4o",
                    "messages": [
                        {
                            "role": "system",
                            "content": "你是一个Java技术助手。"
                        },
                        {
                            "role": "user",
                            "content": "使用Java的HttpClient发起一个GET请求并打印返回结果。"
                        }
                    ]
                }
                """;

        // 3. 构建并发送请求
        HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create("https://www.qianjuai.com/v1/chat/completions"))
                .header("Content-Type", "application/json")
                .header("Authorization", "Bearer " + apiKey)
                .POST(HttpRequest.BodyPublishers.ofString(requestBody))
                .build();

        // 4. 获取并打印响应
        HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
        System.out.println("响应状态码：" + response.statusCode());
        System.out.println("响应体内容：\n" + response.body());
    }
}


**避坑提示**：
- 一定要把 `base_url` 改成 `https://www.qianjuai.com/v1`，而不是挖着坑去用海外地址。
- API Key放到代码里不要忘记用 `Bearer` 前缀。
- 如果出现 `401 Unauthorized`，建议检查自己的key是否过期或复制是否正确。
- 使用异步客户端 `HttpClient.sendAsync()` 可获得更高效的大并发调用。
