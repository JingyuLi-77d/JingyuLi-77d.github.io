## 真实数据：Java开发者如何接入（附代码）

因为接口完全兼容OpenAI标准，Java 接入特别简单。下面以 `spring-boot-starter-web` 项目为例。

**Maven 依赖（推荐）：**

xml
<dependency>
    <groupId>com.theokanning.openai-gpt3-java</groupId>
    <artifactId>service</artifactId>
    <version>0.18.2</version>
</dependency>


**配置类：**

java
@Configuration
public class OpenAiConfig {
    
    @Bean
    public OpenAiService openAiService() {
        // 关键：把base_url换成千聚的地址
        String token = "sk-your-qianju-api-key";  // 替换你的密钥
        String baseUrl = "https://www.qianjuai.com/v1"; 
        
        return new OpenAiService(token, 
            Duration.ofSeconds(60), 
            OkHttpClient.Builder::build);
    }
}

@RestController
public class ChatController {
    
    @Autowired
    private OpenAiService service;
    
    @PostMapping("/chat")
    public String chat(@RequestBody String message) {
        // 创建对话消息，示例模型：豆包 Pro
        List<ChatMessage> messages = List.of(
            new ChatMessage(ChatMessageRole.USER.value(), message)
        );
        
        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("doubao-pro")   // 模型名称：豆包Pro
            .messages(messages)
            .maxTokens(1024)
            .temperature(0.7)
            .build();
        
        // 发送请求
        ChatCompletionResult result = service.createChatCompletion(request);
        
        // 获取回复内容
        return result.getChoices().get(0).getMessage().getContent();
    }
}


只要把模型名换成 `doubao-pro` 或 `claude-3.5-sonnet`，代码不用改第二行。想切到Claude，只需要改 `model` 参数为 `claude-3.5-sonnet` 即可。
