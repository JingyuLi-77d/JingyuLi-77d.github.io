### 第二步：对接GeminiPro（使用OpenAI兼容接口）

java
import okhttp3.*;
import com.fasterxml.jackson.databind.ObjectMapper;
import java.util.*;

public class GeminiProExample {
    // 使用[千聚api中转站](https://www.qianjuai.com/)的OpenAI兼容接口
    private static final String BASE_URL = "https://www.qianjuai.com/v1";
    // 从千聚后台获取的API Key
    private static final String API_KEY = "你的千聚API密钥";

    public static void main(String[] args) throws Exception {
        OkHttpClient client = new OkHttpClient.Builder()
                .connectTimeout(60, TimeUnit.SECONDS)
                .writeTimeout(60, TimeUnit.SECONDS)
                .readTimeout(60, TimeUnit.SECONDS)
                .build();
        ObjectMapper mapper = new ObjectMapper();

        // 构建请求体（兼容OpenAI的chat completions格式，模型名用gemini-2.5-pro）
        Map<String, Object> requestBody = new HashMap<>();
        requestBody.put("model", "gemini-2.5-pro");
        requestBody.put("messages", Arrays.asList(
                Map.of("role", "user", "content", "请用Java实现一个单例模式")
        ));
        requestBody.put("stream", false); // 非流式输出
        requestBody.put("max_tokens", 1000);
        requestBody.put("temperature", 0.7);

        String json = mapper.writeValueAsString(requestBody);
        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .addHeader("Authorization", "Bearer " + API_KEY)
                .post(RequestBody.create(json, MediaType.get("application/json")))
                .build();

        Response response = client.newCall(request).execute();
        if (response.isSuccessful()) {
            String responseBody = response.body().string();
            System.out.println("Gemini返回结果：");
            System.out.println(mapper.readTree(responseBody).at("/choices/0/message/content").asText());
        } else {
            System.err.println("请求失败：HTTP " + response.code() + " - " + response.body().string());
        }
    }
}

1. **模型名称**：千聚文档里对Gemini模型的命名可能和官方略有差异，建议用`gemini-2.5-pro`、`gemini-2.5-flash`、`gemini-1.5-pro`等，如果报错404或400，去千聚文档确认最新模型名。
2. **API Key**：从千聚后台生成的Key请妥善保管，余额不足时会返回401错误，注意检查余额。
3. **代理设置**：因为千聚国内直连，**不需要**在Java程序中设置任何HTTP代理，直接请求即可。如果本地开了全局代理，反而会导致连接失败。
