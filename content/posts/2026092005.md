这是最重要的一步。你只需要改动 `base_url` 和 `api_key` 两条配置，其余代码完全复用 OpenAI API 的标准写法。

python
from openai import OpenAI

# 配置[千聚api聚合平台](https://www.qianjuai.com/)的 API 地址和 Key
client = OpenAI(
    base_url="https://www.qianjuai.com/v1",   # 关键：换成千聚的接口地址
    api_key="sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"  # 替换成你在千聚获取的 API Key

# 调用 o3-mini 模型
response = client.chat.completions.create(
    model="o3-mini",
    messages=[
        {"role": "user", "content": "用 Python 写一个快速排序的代码"}
    ],
    temperature=0.7          # o3-mini 支持调节温度参数
)

# 打印结果
print(response.choices[0].message.content)
