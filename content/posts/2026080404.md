## 为什么聊 Claude 3.5 Sonnet，总绕不开“中转”这两个字？

不是因为中转平台有多厉害，而是官方渠道实在太“劝退”。

**官方直连的几大坑：**

1.  **网络环境硬伤**：国内网络无法直接访问 Anthropic 的 API 端点（`api.anthropic.com`），必须配合科学上网工具。这就意味着你写代码之前，得先“折腾”完网络。
2.  **付款门槛**：Anthropic 要求绑定海外发行的 Visa/Mastercard。没有一张国际信用卡？或者绑卡时因为风控被拒？对不起，注册不了。
3.  **账户风控**：官方对账户活跃度、IP 纯净度有要求。一个新账户，静置几天或者 IP 共享，可能就直接被“封号静默”了。而你充进去的预付款，通常不退。
4.  **配额限制**：官方 API 有速率限制（Rate Limit），对个人开发者和小型项目来说，频繁调用很容易触顶。

相比之下，**三方中转平台**（比如千聚 api 聚合平台）解决的正是这几个核心痛点：国内直连、支付宝/微信支付、低门槛充值、无脑兼容。

### 方案一：官方直连“大冒险”

**总共耗时：约 45分钟（主要是卡在绑卡和配置代理上）**

1.  **注册 Anthropic 账号**：必须用科学上网工具，访问 `console.anthropic.com`。
2.  **绑卡**：最痛苦的环节。我先是试了几张国内发行的双币信用卡，全部被拒。最后借了张朋友的美国虚拟信用卡才通过。
3.  **创建 API Key**：在 Dashboard 里生成密钥。
4.  **配置代码调用**：用最常规的 `requests` 库调用 Anthropic 原生接口，或者用 `anthropic` Python SDK。需要处理官方的特定 header 和消息格式。

python
# 官方直连代码示例（需要配置代理）
import anthropic

client = anthropic.Anthropic(
    api_key="sk-ant-your-key",
    proxies={"http": "http://127.0.0.1:7890", "https": "http://127.0.0.1:7890"} # 每个人代理配置不同
)

message = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1024,
    messages=[{"role": "user", "content": "用Python写一个快速排序"}]
)
print(message.content[0].text)
